🇮🇩 Game Kuis Interaktif · Kelompok 4 🇮🇩

⚔️ Mode 1 — Rebut Bendera (2 Pemain)
  • Layar dibagi 2 — Kiri (Merah) vs Kanan (Biru)
  • Karakter bergerak maju setiap jawaban BENAR
  • Yang pertama merebut bendera di tengah = MENANG
  • Waktu 70 detik per pemain. Benar −3 dtk, Salah −5 dtk
  • Soal salah masuk antrian ulang sampai dijawab benar
🤝 Mode 2 — Tim Battle (Maks 20/Tim)
  • 2 Tim bersaing — Tim Merah (kiri) vs Tim Biru (kanan)
  • Peserta bergantian menjawab soal dalam tim masing-masing
  • Wakil tim bergerak ke bendera setiap jawaban benar
  • Suporter menonton & bersemangat di pinggir lapangan
  • Setup: masukkan nama Tim Merah dulu → Siap → Tim Biru
☕ Cozy Mode — Solo Santai
  • Untuk 1 pemain, tanpa batas waktu sama sekali
  • Jawab 20 soal sambil belajar dengan santai
  • Karakter berjalan santai setiap jawaban benar
  • Cocok untuk belajar mandiri tanpa tekanan
🎵 Sistem Suara
  • Musik latar (musik.mp3) dimainkan di lobi
  • Efek suara otomatis mengecilkan musik (ducking)
  • Toggle suara tersedia di beranda
🛠️ Cara Pembuatan Game
  • Teknologi: HTML5, CSS3, JavaScript murni (vanilla JS) — tanpa library/framework
  • Canvas API: Arena permainan digambar dengan HTML Canvas untuk animasi 60fps
  • Web Audio API: Efek suara & musik dibuat langsung dari oscillator + ducking
  • PWA: Progressive Web App — bisa diinstal & fullscreen di HP
  • Responsive: Mendukung portrait/landscape, HP, tablet, dan TV
⚙️ Fitur Teknis
  • Animasi karakter 60fps dengan requestAnimationFrame
  • Physics fall — karakter jatuh dramatis saat kalah
  • Karakter manusia realistis dengan kulit, rambut, ekspresi
  • Sistem timer independen untuk setiap pemain/tim
  • Auto-save state untuk fitur Resume
  • Batik pattern sebagai dekorasi latar belakang
👥 Tim Pengembang

  • Dibuat untuk keperluan pembelajaran interaktif
  • Guru: Bapak/Ibu Informatika SMP
