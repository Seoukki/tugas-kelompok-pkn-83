Tentang Game

Kuis Battle adalah game kuis interaktif berbasis web yang dibuat oleh Kelompok 4 untuk meningkatkan pemahaman Pengetahuan Umum Indonesia. Game ini berjalan sepenuhnya di browser tanpa instalasi apapun.

🛠️

Cara Pembuatan

Game ini dibuat menggunakan teknologi web murni:

  • HTML5 — Struktur halaman dan elemen UI
  • CSS3 — Animasi, layout, dan tampilan visual dengan CSS custom properties
  • JavaScript (Vanilla) — Logika game, canvas rendering, audio engine
  • Canvas API — Animasi arena, karakter, tali, dan efek visual real-time
  • Web Audio API — Efek suara synthesizer tanpa file audio eksternal
  • Google Fonts — Font Bebas Neue (display) dan Plus Jakarta Sans (body)
📏

Aturan Mode 1 — Tarik Tambang

  • 2 pemain bermain bersamaan di layar yang sama (split screen)
  • Setiap pemain mendapat 70 detik untuk menjawab 20 soal
  • Jawaban benar: waktu berkurang 3 detik, karakter maju
  • Jawaban salah: waktu berkurang 5 detik, soal masuk antrian lagi
  • Menang jika menyelesaikan semua 20 soal lebih dulu, atau lawan kehabisan waktu
  • Karakter yang kalah akan jatuh secara dramatis di arena!
🤝

Aturan Mode 2 — Tim Battle

  • 2 tim (max 20 orang per tim) bertanding bersamaan
  • Soal dijawab bergantian oleh anggota tim (giliran otomatis)
  • Setiap giliran pemain mendapat 70 detik
  • Sistem poin sama dengan Mode 1 (benar -3s, salah -5s)
  • Tim Oranye vs Tim Biru — nama tim tampil di arena
  • Tim pertama selesaikan 20 soal = MENANG!
🌿

Aturan Mode 3 — Cozy Mode

  • Solo player, tanpa timer, tanpa tekanan
  • Jawab 20 soal, karakter berjalan menuju garis finish
  • Setiap jawaban (benar/salah) menampilkan penjelasan edukatif
  • Tidak ada hukuman — semua jawaban memajukan permainan
  • Cocok untuk belajar santai dan memahami materi
💡

Fitur Khusus

  • 🎵 Musik latar (musik.mp3) dengan volume otomatis saat ada efek suara
  • ▶ Tombol Resume — lanjutkan game yang tertinggal
  • 📋 Lihat semua soal beserta penjelasan setelah permainan
  • 🎭 Karakter animasi real-time dengan ekspresi berbeda
  • 👥 Animasi penonton yang bersorak saat tim memimpin
  • 💥 Efek jatuh dramatis bagi pihak yang kalah
  • 🌐 Responsive — bisa dimainkan di HP maupun laptop
